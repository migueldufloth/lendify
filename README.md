Sistema web para gerenciamento de empréstimo e reserva de equipamentos como projetores, notebooks e kits.

## Stack

- Front-end: React
- Back-end: Django (Python)
- Banco de dados: MySQL

## Perfis de usuário

- Solicitante: realiza e acompanha reservas próprias
- Operador: aprova, recusa e registra devoluções
- Administrador: gerencia equipamentos, usuários e logs
